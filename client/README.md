# Beauty Parlour Template — Client (Frontend)

React + Vite + Tailwind CSS frontend.

## Quick Start
```bash
npm install
npm run dev
```

## Key files to edit:
- `src/config/site.js` — Business info (name, phone, email, social links)
- `src/config/theme.js` — Colors, fonts, border radius
- `src/data/` — Services, pricing, team, testimonials, FAQs, gallery
- `src/index.css` — Font imports
- `public/` — Replace all images

See the root `TEMPLATE_GUIDE.md` for full customization instructions.
